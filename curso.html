Detalhes do Curso

Formação Completa

O curso técnico em Desenvolvimento de Sistemas tem duração de 3 anos, integrado ao ensino médio, preparando estudantes para o mercado de tecnologia desde cedo.

Grade Curricular

  • Lógica de Programação
  • Desenvolvimento Web (HTML, CSS, JavaScript)
  • Programação em Java e Python
  • Banco de Dados e SQL
  • Desenvolvimento Mobile
  • Redes de Computadores
  • Engenharia de Software

Oportunidades Profissionais

Os egressos do curso podem atuar como: Desenvolvedores Web, Programadores Mobile, Analistas de Sistemas, Administradores de Banco de Dados, entre outras funções no mercado de tecnologia.
